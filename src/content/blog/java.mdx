 # Primeros paos en el leguaje java


## Sintaxis básica

Java está compuesto por elementos, cada uno con una función específica. Por ejemplo, la clase `public` y el método `static void main(String[] args)` son parte de la estructura de un programa en Java.

## Elementos básicos

Aquí tienes algunos elementos básicos de Java:

- Clase
- Método
- Atributo
- Comentarios: `//` para comentarios en una línea y `/* */` para comentarios en varias líneas.
- Declaración de variables: `TipoNombreVariable = ValorInicial;`
- Tipos de datos: `int`, `float`, `double`, `boolean`, `String`, `char`, `byte`, `short`, `long`.
- Operadores aritméticos: `+`, `-`, `*`, `/`, `%`.
- Operadores relacionales: `==`, `!=`, `>`, `<`, `>=`, `<=`.
- Operadores lógicos: `&&` (AND), `||` (OR), `!` (NOT).
- Condicionales: `if`, `else`, `else if`.
- Bucles: `for`, `while`.
- Métodos: `public`, `private`, `static`, `void`.
- Atributos de clase: `public`, `private`, `static`, `final`.
