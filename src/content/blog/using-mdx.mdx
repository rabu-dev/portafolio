## Introducción

¿Qué es HTML?
HTML es el lenguaje de marcado para la creación de páginas web. Permite describir la estructura de una página web y su contenido.

## Sintaxis básica

HTML está compuesto por elementos, cada uno con una función específica. Por ejemplo, el elemento `<h1>` se utiliza para representar un encabezado de nivel 1 en una página web.

## Elementos básicos

Aquí tienes algunos elementos básicos de HTML:

- `<a>` para enlaces
- `<img>` para imagenes
- `<h1>` a `<h6>` para encabezados
- `<p>` para párrafos
- `<ul>` y `<ol>` para listas sin orden y orden respectivamente
- `<li>` para elementos de lista
- `<strong>` y `<em>` para texto en negrita y cursiva
- `<blockquote>` para bloques de cita
- `<code>` para código en línea
- `<div>` para divisiones
- `<span>` para divisiones en línea
- `<main>` para contenido principal
- `<header>` y `<footer>` para encabezado y pie de página respectivamente
- `<nav>` para navegación
- `<article>` para artículos
- `<section>` para secciones
- `<aside>` para contenido lateral
- `<video>` y `<audio>` para multimedia
- `<table>` para tablas
- `<thead>` para el encabezado de la tabla
- `<tbody>` para el cuerpo de la tabla
- `<tfoot>` para el pie de página de la tabla
- `<tr>` para filas de la tabla
- `<th>` para celdas de encabezado
- `<td>` para celdas de datos
